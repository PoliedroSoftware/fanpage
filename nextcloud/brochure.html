¿Qué es Nextcloud?

Nextcloud es la plataforma de código abierto líder mundial para nube privada. Ofrece todas las funcionalidades de Dropbox, Google Drive y Office 365 combinadas en una sola plataforma que tú controlas. Tus archivos residen en tus servidores, con las máximas garantías de seguridad, privacidad y cumplimiento normativo.

💻

Desktop Client

Sincronización nativa para Windows, macOS y Linux. Archivos siempre disponibles en tu explorador, incluso sin conexión a internet.

📱

App Móvil

iOS y Android. Subida automática de fotos, acceso offline, compartición instantánea. Tus archivos siempre en tu bolsillo.

🌐

Acceso Web

Interfaz completa desde cualquier navegador. Arrastra y suelta archivos, edita documentos y gestiona permisos.

Características

Todo lo que Nextcloud puede hacer por tu organización

  • Sincronización selectiva: elige qué carpetas sincronizar en cada dispositivo.
  • Compartición segura: enlaces con contraseña, fecha de expiración y permisos granulares.
  • Versionado ilimitado: recupera cualquier versión anterior de un archivo.
  • Cifrado E2E: carpetas con cifrado de extremo a extremo.
  • Collabora Online: edita documentos, hojas de cálculo y presentaciones en equipo.
  • Calendario y Contactos: sincronización CalDAV/CardDAV en todos tus dispositivos.
  • Nextcloud Talk: videollamadas, chat y compartir pantalla cifrados.
  • Nextcloud Mail: cliente de correo integrado en la interfaz web.
  • Dashboard personalizable: widgets de clima, correo, tareas y más.
  • LDAP / Active Directory: integración con tu directorio corporativo.
  • SAML / OIDC / MFA: autenticación empresarial segura.
  • External Storage: monta S3, FTP, NFS, SharePoint como carpetas locales.
  • API REST y WebDAV: integración con tus sistemas existentes.
  • Flujo de trabajo y automatizaciones: reglas para aprobaciones, notificaciones y más.
  • Auditoría completa: registro de cada acceso, modificación y compartición.
  • Marketplace de apps: +300 aplicaciones para extender funcionalidades.
